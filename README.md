# MoarFX

MoarFX is a mod adding more visual effects to KSP.

---

Current Features:

- Kicked Up Dust When Driving
- Sparks (Decoupling and Scraping)

---

Planned Features:

- Engine Contrails
- Touchdown Smoke (Finished but looks horrible, not releasing it for a bit)
- Micro Debris in Orbit
- Heatshield Separation "Flashes"

---

Known issues:

- Kicked up dust is still active anywhere outside of the stock KSC runway
- When Kerbals run sparks are shown
